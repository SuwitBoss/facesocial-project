# หน้า CCTV Monitoring (Admin Features)

## ภาพรวม
ระบบติดตาม CCTV ขั้นสูงสำหรับผู้ดูแลระบบ รองรับการดูกล้องแบบเรียลไทม์ การตรวจจับใบหน้าอัตโนมัติ การแจ้งเตือนความผิดปกติ และการวิเคราะห์พฤติกรรมด้วย AI

## การเข้าถึง
- **สิทธิ์**: เฉพาะ Admin เท่านั้น
- **การยืนยันตัวตน**: ต้องผ่าน 3-step Face Login
- **Session Timeout**: 30 นาที (ด้วยเหตุผลความปลอดภัย)

## หน้าในระบบ

### 1. หน้า CCTV Dashboard หลัก
**URL**: `/admin/cctv` หรือ `/monitoring/dashboard`

### 2. หน้าดูกล้องแบบเรียลไทม์
**URL**: `/admin/cctv/live/:cameraId`

### 3. หน้าการตรวจจับใบหน้า
**URL**: `/admin/cctv/face-detection`

### 4. หน้าการแจ้งเตือน
**URL**: `/admin/cctv/alerts`

### 5. หน้าบันทึกและประวัติ
**URL**: `/admin/cctv/recordings`

### 6. หน้าการตั้งค่า CCTV
**URL**: `/admin/cctv/settings`
